PROJECT OVERVIEW

ConvoAI is a 6-week MVP development project focused on building an AI-powered Spanish conversation practice web application. The project is organized into three sprints, each with clearly defined goals, deliverables, and milestones. This schedule is derived directly from the project Work Breakdown Structure (WBS) and is designed to be realistic for a 3-person student team.
